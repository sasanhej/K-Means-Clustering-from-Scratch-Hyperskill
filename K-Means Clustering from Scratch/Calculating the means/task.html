Description

The next step of the K-Means algorithm is to calculate new coordinates for each cluster's center once you know which of the "old" clusters every object belongs to. To do it, you need to calculate each feature's mean value for each cluster's objects â€” and name it the "new" cluster center.

Name the new function calculate_new_centers. This new function, together with the previous one, would be later wrapped into a class. You could either implement it into the class now or make it a standalone function. You get to decide whatever feels better to you.

Objectives
  1. Create the calculate_new_centers function and calculate new centers as the mean features for each of the classes;
  2. Use the labels you got in the previous stage (using the find_nearest_center function) to calculate new centers;
  3. Print the resulting coordinates of the centers. Use the .flatten() method to turn a 2D array into 1D.
