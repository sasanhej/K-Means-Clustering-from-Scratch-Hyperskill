Description

In this stage, let's challenge your code. Use optimal k you found in the previous steps and your custom K-Means class to create the predictions on clusters for each object in X. Then, you can use our function to plot your predictions (one 2D subdimension would do) or make yours. We asked you to guess how many clusters would be required for a great fit. It is now a perfect time to see whether you were correct.

The function we provided actually plots two images: one is your predictions, and the second one is the benchmark data so you can compare those and estimate how well your model performs.
